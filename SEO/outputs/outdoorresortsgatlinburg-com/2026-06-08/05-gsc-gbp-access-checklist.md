# GSC and GBP Access Checklist

Date: 2026-06-08

## Google Search Console

Google's Search Console documentation recommends a Domain property when possible because it combines protocol and subdomain variants. For this site, that means verifying:

- Domain property: `outdoorresortsgatlinburg.com`
- Sitemap URL: `https://outdoorresortsgatlinburg.com/sitemap.xml`

### Steps

1. Open Google Search Console.
2. Add a Domain property for `outdoorresortsgatlinburg.com`.
3. Verify ownership, ideally with DNS TXT verification.
4. Open the Sitemaps report.
5. Submit `https://outdoorresortsgatlinburg.com/sitemap.xml`.
6. Inspect these priority URLs:
   - `https://outdoorresortsgatlinburg.com/`
   - `https://outdoorresortsgatlinburg.com/rentals.html`
   - `https://outdoorresortsgatlinburg.com/stay-guide.html`
   - `https://outdoorresortsgatlinburg.com/rocky-top-sports-world.html`
   - `https://outdoorresortsgatlinburg.com/investing-units-for-sale.html`
7. Export the first baseline after data appears:
   - Queries
   - Pages
   - Countries/devices
   - Indexing status
   - Sitemap status
